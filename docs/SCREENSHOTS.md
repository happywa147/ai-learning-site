# 截图与传播素材说明

这个项目需要几张稳定截图，用于 GitHub README、社交媒体介绍、版本更新说明和用户反馈收集。

## 建议截图清单

### 1. 首页与学习驾驶舱

用途：README 顶部、社交分享首图。  
建议文件名：`docs/images/home-dashboard.png`

画面应包含：

- 网站标题
- 主要行动按钮
- 右侧学习驾驶舱视觉
- 下方学习总览数据的一部分

### 2. 60 秒新手上手

用途：说明项目对新人友好。  
建议文件名：`docs/images/quick-start.png`

画面应包含：

- 三步新手任务
- “选路线 / 做挑战 / 看样例”的入口
- 枣红、暖土、绿植色系整体风格

### 3. 每月更新区

用途：突出“不是固定教材，而是持续更新”。  
建议文件名：`docs/images/monthly-update.png`

画面应包含：

- 月份选择器
- 本月主题
- 模型观察 / 市场观察 / 本月训练 / 避坑提醒

### 4. 作品样例库与反馈区

用途：突出作品导向和社区共创。  
建议文件名：`docs/images/showcase-feedback.png`

画面应包含：

- 作品样例卡片
- 反馈表单
- 反馈导出入口

## 截图尺寸建议

- README 宽图：`1600 x 900`
- 社交平台横图：`1200 x 630`
- 移动端截图：`390 x 844`

## 视觉要求

- 使用真实页面截图，不使用模糊或过暗背景。
- 页面缩放保持默认 100%。
- 不截取个人二维码或赞助二维码作为项目宣传主图。
- 保留页面主体内容，不要只截顶部导航。
- 如果包含反馈或报名数据，请使用测试数据。

## README 嵌入方式

截图准备好后，可在 README 中加入：

```md
![首页与学习驾驶舱](./docs/images/home-dashboard.png)
```

如果图片还没有准备好，README 先只保留截图说明链接，避免出现损坏图片。
