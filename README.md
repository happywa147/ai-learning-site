# AI 原生能力自学站

一个面向 AI 学习者、创作者、职场人和项目实践者的自学网站。

它不是一本固定教材，而是一个可以持续更新的 AI 学习驾驶舱：通过任务、等级、徽章、作品项目、月度更新和世界观框架，帮助学习者长期跟上 AI 应用市场的变化。

## 核心功能

- AI 学习游戏化：签到、XP、等级、徽章和任务进度
- AI 时代世界观：基础设施、协作方式、验证精神、责任边界
- 每月更新区：模型观察、中国应用市场、本月训练、避坑提醒
- 三条学习路线：零基础入门、短视频创作者、AI 应用开发
- 国内外模型对比：按应用场景理解模型差异
- 作品集项目：学习助教、短视频脚本工厂、知识库问答、Agent 小工具等
- Prompt、Agent、Skill 模板库
- 注册与联系：本地保存报名信息，可导出 CSV
- 赞助窗口：支持微信扫码赞助

## 本地运行

这是一个纯静态网站，不需要构建工具。

```bash
python3 -m http.server 8765
```

然后打开：

```text
http://localhost:8765
```

也可以直接打开 `index.html` 预览，但使用本地服务器更接近真实部署环境。

## 数据与隐私说明

- 学习进度、签到、徽章、报名信息保存在浏览器 `localStorage` 中。
- 当前静态版本不会把报名信息自动发送到服务器。
- 邮箱在页面上不直接明文展示，而是通过点击按钮复制。
- 微信联系二维码和赞助二维码分别作为图片资源保存在 `assets/` 目录。

## 维护方式

每月更新内容位于 `app.js` 的 `monthlyUpdates` 数组中。

新增一个月份时，复制已有结构并更新：

- `label`
- `title`
- `summary`
- `cards`

模型对比、路线、项目和模板也都集中在 `app.js` 中，方便后续迭代。

## 部署建议

可以直接使用 GitHub Pages 部署：

1. 进入仓库 Settings
2. 打开 Pages
3. Source 选择 `Deploy from a branch`
4. Branch 选择 `main`，目录选择 `/root`
5. 保存后等待 GitHub Pages 生成访问地址

## 项目定位

AI 变化太快，固定教材很容易过时。这个网站的目标是把“学习 AI”做成一个长期系统：

- 底层学世界观和判断力
- 中层学工具、模型和工作流
- 上层跟进中国应用市场与真实项目
- 最终用作品证明学习结果
