# 贡献者公约（Code of Conduct）

## 我们的承诺

为了营造一个开放友善的环境，我们作为贡献者和维护者承诺：无论年龄、体型、可见或不可见的残疾、族裔、性别认同与表达、经验水平、教育、社会经济地位、国籍、个人外貌、种族、宗教或性取向，参与本项目和社区的每一个人都将免受骚扰。

我们承诺以促进开放、包容、多样化、健康社区的方式行动和互动。

## 我们的准则

营造正面环境的行为示例：

- 对他人展示同理心和善意
- 尊重不同意见、观点和经验
- 给予建设性反馈，也优雅地接受反馈
- 承担责任并向受我们错误影响的人道歉，从中学习
- 关注的不仅是个人，更是整个社区的福祉

不可接受的行为示例：

- 使用性化语言或意象，以及任何形式的性骚扰或挑逗
- 钓鱼、侮辱或贬损性评论，以及人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人的私人信息（如物理地址或电子邮箱）
- 其他可被合理认定为不适合专业环境的行为

## 维护者责任

项目维护者有责任澄清和执行本公约的标准，并将对其认为不当、威胁、冒犯或有害的任何行为采取适当且公平的纠正措施。

维护者有权删除、编辑或拒绝不符合本公约的评论、提交、代码、wiki 编辑、议题和其他贡献，并可在必要时说明理由。维护者需公平一致地执行这些标准。

## 适用范围

本公约适用于项目内的所有社区空间，也适用于个人在公共空间中正式代表项目时的行为。

代表项目的情况包括但不限于：使用官方项目电子邮件地址、通过官方社交媒体账号发帖、在线上或线下活动中担任指定代表等。项目的代表规则可由维护者进一步明确。

## 执行

辱骂、骚扰或其他不可接受的行为可通过私下联系项目维护者举报：在 GitHub 仓库创建 Issue 并标记为 `conduct`，或发送邮件至维护者邮箱。所有投诉都会被及时、公正地审查和调查。

项目维护者有义务尊重举报者的隐私和安全。
